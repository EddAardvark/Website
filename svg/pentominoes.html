SVG Pentominoes

The 17 Pentominoes

We have 17 because we regard the mirror images as different examples. Ignoring these there would only be 12. The images are defined in the file pentominoes.svg coupled with the stylesheet pentominoes.css.

Below I render each one as a separate SVG object using the viewbox attribute to map the object's co-ordinate space (-50,50) to (50,50) to the screen's (0,0) to (100,100).
