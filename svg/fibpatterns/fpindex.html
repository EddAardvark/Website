Original Fibonacci Patterns Page

These patterns are generated by generating Fibonacci sequences in modulo arithmetic and the plotting consecutive pairs of values as co-ordinates. (See the Algorithm page for more details.)

The patterns on this page are mainly hand drawn and show some simple examples.

If you browser supports Googls's native client plugin you can use the new interactive version. This will allow you to create more complicated examples, up to 600x600, and create animations by varying the size or starting parameters.

The examples on this page have (0,0) in the bottom left, the plugin has (0,0) in the top left.
