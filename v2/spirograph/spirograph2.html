Spirographs

"Spirograph" is a registered trademark of Hasbro, Inc. It refers to a child's toy that uses toothed wheels to generate patterns, you can see some examples here.

This page draws spirograph patterns, both the traditional 2 wheel versions and the more generic "epicycles" using 3 or more wheels. You can use the various controls to manipulate the patterns and to create animations.

Instructions

All the action takes place in the area labelled "Play Area". The sub-sections within this areas are described below.

Configure

This area contains an array of buttons that act like a tab bar. Each button will display one of the confiuration dialogs, which will appear to the right of the patter.

You can get more information about what each tab bar dialog does by hovering your pointer over the corresponding button, where a popup window will appear.

Display

Below the toolbar are 6 buttons that select the type of pattern that is drawn in the "Pattern Area", immediately below these buttons. You can get more information about what each mode looks ilke by hovering the mouse over the button.

Actions

Alongside the display buttons there are some additional buttons for various functions. Hover over the individual buttons for descriptions.

Pattern Area

This appears on the left, directly below the display buttons. This is where the result of your configuration choices appears.

Dialog Area

The contents of this area change depending on the selected tab in the tab bar area. Each instance contains a set of controls that manage some aspect of the pattern that is drawn. You can get more information on these controls by hovering over the information icon that appears towards the top right of this area. Try it with the example icon below:

Status Area

Currently this shows the values of the parameters that describe the basic pattern. I may expand it to include the split and overlay attributes in the future. When the pattern is animated these values will update.
