The Galleries

Currently there are only 4 gallery types but I intend to add more. They show random examples of the patterns that can be generated using to other pages on this site.

Entrance

You will start of in the garden. This image shows the entrance to a gallery, there are also a number of animated sculptures, maps and mazes scattered around..

Triangle Patterns

This starts out with a collection of up to 4 random triangles, then expands the pattern by applying a number of random symmetry operations.

Rectangle Patterns

This starts out with a collection of up to 4 random rectangles, then expands the pattern by applying a number of random symmetry operations.

Mixed Triangles and Rectangles

This starts out with a collection of up to 4 random elements, including rectangles and triangles, then expands the pattern by applying a number of random symmetry operations.

Cellular Automata

This generated random cellular automata based on the five colour algorithm described elsewhere on this site. If you view one of the images in close up it will provide you with a link the the automata home page and a code that will enable you to recreate and interact with the image you are currently viewing.

Or go directly to the page.

Maze

This view is from the inside of one of the mazes, there are currently three types. You can see one of the animated sculptures in the background.
