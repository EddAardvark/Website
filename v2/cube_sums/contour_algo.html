How the algorithm works

We start where the line x^3+y^3-(x+n)^3=0 intersects x=y which gives 2x^3=(x+n)^3, or x=n/(2^[1/3]-1) (approx n/0.26), which we round up to the next integer.

Next we create a class the allows us to track y cubed. We start by calculating the difference between adjant cubes, then the differences between the differences and then the differences between those until we arrive at a constant. If we start at 1, this looks like:

V182764125
ΔV 7193761
ΔΔV  121824
ΔΔΔV   66

To calculate the next cube we start at the bottom, and extend the row right by inserting a 6. We then move up the new column by adding 6 to the last value, 24, giving 30. The second row becomes 61+30=91 and the top row 125+91=216 (6^3). More generically:

V y^3 (y+1)^3 (y+2)^3 (y+3)^3
ΔV   3y^2+3y+1 3y^2+9y+7 3y^2+15y+19
ΔΔV     6y+6 6y+12
ΔΔΔV       6
