Higher order Mandelbrot style patterns

This page shows the patterns obtained by using higher order versions of the Mandelbrot generating polynomial, z -> zn + c. There are three obvious trends as the value of 'n' increases:

  1. The symetry of the pattern increases, each pattern has (n-1) planes of reflection symetry and (n-1)-fold rotational symetry.
  2. The area inside the set, coloured black, increases, the central region aproaching the unit circle.
  3. The contours outside the black regions become closer together (points not inside the set escape very quickly) so there is less of the interesting spiral filigree observed when zooming in, compared with the original (n=1) Mandelbrot pattern.
