Mandelbrot Creator Colour Schemes

Introduction

There are 8 colour schemes, the one that is used will be determined by the −d parameter passed to the MandelbrotDriver. If you don't choose a colour scheme then the default scheme for the pattern sub-type you have choses (Standard, Stars, Bands, etc.) will be used.

An example of each scheme is shown below.

Parameter Appearance Description
−d1 Image drawn with the standard sub-type, −M.
−d2 Image drawn with the standard sub-type, −M.
−d3 Image drawn with the "Stars" sub-type, −MS.
−d4 Image drawn with the "Bands" sub-type, −MB.
−d5 Image drawn with the "Bands" sub-type, −MB.
−d6 Image drawn with the "Quadrants" sub-type, −MQ.
−d7 Image drawn with the "Stars" sub-type, −MS.
−d8 Image drawn with the standard sub-type, −M.
